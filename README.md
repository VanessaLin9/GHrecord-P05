### Feture-專案功能
***
#### 使用者可以:
+ 紀錄每次參與的遊玩者
+ 紀錄每次遊戲的關主
+ 抽選或指定當次的遊戲關主
+ 顯示過往遊戲紀錄
+ 刪除過往遊戲紀錄
+ 紀錄當次遊玩的關卡
+ 觀看隊伍中參與者
+ 更改隊伍中參與者扮演的角色
+ 更改隊伍中參與者的名字
+ 新增隊伍參與者
